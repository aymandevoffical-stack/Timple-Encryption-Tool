# 🔐 Simple Text Encryption Tool

A lightweight web-based tool developed to demonstrate basic encryption logic using **JavaScript**. This project is part of my technical portfolio for scholarship applications in **Cybersecurity / Computer Science**.

## 🚀 How it Works
This tool uses a custom **Caesar Cipher** logic. It shifts the character codes of the input text to transform readable messages into encrypted strings and vice versa.

## 🛠️ Features
- **Encryption:** Converts plain text into secure symbols.
- **Decryption:** Reverses the process to retrieve the original message.
- **Clean UI:** Responsive design built with HTML5 and CSS3.

## 💡 Purpose
The goal of this project was to practice:
1. Data manipulation using JavaScript.
2. Understanding the basics of Cryptography.
3. Building functional web tools from scratch.
